> sound for cli

## Install

```
$ yarn add symphogear
```

## Usage

```js
const symphogear = require('symphogear')

await symphogear('music.mp3')
```

## API

### `symphogear(file: string, opts?: {player?: string})`

#### file

Type: `string`

support file or url.

#### options

##### player

Type: `string`<br>

### Available audio players to check.

- mplayer
- afplay
- mpg123
- mpg321
- play
- omxplayer
- aplay
- cmdmp3
